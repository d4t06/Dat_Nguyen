## CPU

### Intel

_Core i[3-5-7-9] [Generation] [Suffix]_

**Suffix**

- (No suffix): Standard desktop processor.

- K: CPU can be easily overclocked (e.g., i7-14700K).

- F: No integrated graphics (requires a dedicated graphics card) (e.g., i3-12100F).

- KF: Unlocked and no integrated graphics.

- T: Lower TDP (e.g., i7-14700T).

- U: Laptop, battery save (i3-10110U).

- Y: Laptop, very low power (i5-10310Y).

- G[1-7]: Like U, enhanced integrated graphics (Iris Xe).

- H: Laptops, high performance (e.g., i7-13700H).

**Sockets**

- LGA 1700 (Current Mainstream): 12th Gen (Alder Lake), 13th Gen (Raptor Lake), 14th Gen (Raptor Lake Refresh)

- LGA 1200: 10th Gen (Comet Lake) and 11th Gen (Rocket Lake)

- LGA 1151 v2 (or "LGA 1151-2"): 8th Gen (Coffee Lake) and 9th Gen (Coffee Lake)

- LGA 1151 : 6th Gen (Skylake) and 7th Gen (Kaby Lake).

- LGA 1150: 4th Gen (Haswell) and 5th Gen (Broadwell)

### AMD

_Ryzen [3-5-7] [Generation] [Suffix]_

**Generation**

- Generation: 3000, 4000, 5000... (Zen, Zen+, Zen 2, Zen 3, Zen 3+, Zen 4...)

**Suffix**

- (No suffix): Standard desktop processor (no integrated graphics)

- G: With integrated graphics

- X: Higher-performance

- F(rare): No integrated graphics

**Socket**

- AM4: 5000 below

- AM5: 7000 above

### Tốc độ theo giao thức

|                 | Read                  | Write                 | Socket                             |
| :-------------- | :-------------------- | :-------------------- | :--------------------------------- |
| **SATA III**    | Up to 550 MB/s        | Up to 500 MB/s        | 2.5-inch, M2 Sata (B+M key), mSATA |
| **PCIe 3.0 x4** | 2,500 - 3,500 MB/s    | 1,500 - 3,000 MB/s    | M2 (M key)                         |
| **PCIe 4.0 x4** | 5,000 - 7,000+ MB/s   | 4,000 - 7,000+ MB/s   | M2 (M key)                         |
| **PCIe 5.0 x4** | 10,000 - 14,000+ MB/s | 10,000 - 14,000+ MB/s | M2 (M key)                         |

## PCIe speed per lane

- PCIe 1.0 (2003): 250 MB/s.
- PCIe 2.0 (2007): 500 MB/s.
- PCIe 3.0 (2010): ~985 MB/s (approximately 1 GB/s).
- PCIe 4.0 (2017): ~1.97 GB/s.
- PCIe 5.0 (2019): ~3.94 GB/s.

## SATA Operations

### AHCI

- Advance Host Controller Interface: The most use of modern pc

### RAID

- Redundant Array of Independent Disks: Combine multiple drives into a single unit for improved performance, redundancy, or both.

| Feature             | RAID 0 (Striping)                                                                                                          | RAID 1 (Mirroring)                                                                                                             | RAID 5 (Striping with Parity)                                                                               |
| :------------------ | :------------------------------------------------------------------------------------------------------------------------- | :----------------------------------------------------------------------------------------------------------------------------- | :---------------------------------------------------------------------------------------------------------- |
| **Minimum Drives**  | 2                                                                                                                          | 2                                                                                                                              | 3                                                                                                           |
| **Data Redundancy** | None                                                                                                                       | Full Redundancy (mirrored copy)                                                                                                | Single-drive redundancy (distributed parity)                                                                |
| **Data Protection** | If any one drive fails, all data is lost.                                                                                  | Can withstand the failure of one drive.                                                                                        | Can withstand the failure of one drive.                                                                     |
| **Performance**     | Improve read and write speeds scales with the number of drives.                                                            | Improve read speed, write speed acts as a single drive or slightly slower.                                                     | Improve read speed. Write speed is slower due to parity calculations.                                       |
| **Usable Capacity** | 100% of total drive capacity (sum of all drives).                                                                          | 50% of total drive capacity.                                                                                                   | (n-1) drives, where 'n' is the total number of drives.                                                      |
| **Best Use Case**   | Situations where speed is the only priority and data loss is not a concern, such as video editing scratch disks or gaming. | Mission-critical data where high availability and reliability are paramount, like for operating systems or accounting systems. | General-purpose servers and applications that need a balance of performance, capacity, and data protection. |

- Intel Optance acts as a high-speed cache between a computer's processor and slower storage devices like hard drives (HDDs) or solid-state drives (SSDs

- Use for accelerating a single drive

- Require 7th gen or newer

- All modern SATA and NVMe SSDs use 3D NAND flash memory technology, Intel Optane use special technology called 3D Xpoint, achieve higher speed

## Intel RST

- Is the software that provides the drivers and management tools for both Optane memory and RAID configurations.

#### B, M Keys

- M Key (PCIe x4): NVMe SSDs

- B Key (PCIe x2): Used for SATA, SSDs WWAN (Wireless Wide Area Network)

- B + M Key (PCIe x2 - Module only): Thiết kế để cắm module vào B key hoặc M key điều được

#### A, E Keys

<CustomImage

- A Key (PCIe x2): Used for Wi-Fi, Bluetooth, WWAN, GPS, and NFC cards.

- E Key (PCIe x2): Often used for Wi-Fi, Bluetooth

- A + E Key (Module only): Thiết kế để cắm module vào socket A key hoặc E key điều được, thường là các card Wi-Fi tích hợp Bluetooth.
